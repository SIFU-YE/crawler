#### 通过
# MDX 数据定义 - DROP SET
删除一个命名集。
```
  
DROP [SESSION] SET   
   <Cube_Reference>.Set_Name   
               [,<Cube_Reference>.Set_Name ...n]  
  
<Cube_Reference> ::= {CURRENTCUBE | Cube_Name}  
  

```

_Cube_Name_ 提供多维数据集名称的有效字符串表达式。
_Set_Name_ 有效字符串表达式，提供将要删除的命名集的名称。
有关命名集的详细信息，请参阅 在 MDX (MDX) 中生成命名集 。
