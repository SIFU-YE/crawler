#### 通过
# MDX 数据定义 - DROP MEMBER
删除一个计算成员。
```
  
DROP MEMBER   
   CURRENTCUBE | Cube_Name  
      .Member_Name   
               [,CURRENTCUBE | Cube_Name.Member_Name ...n]  

```

_Cube_Name_ 提供多维数据集名称的有效字符串表达式。
_Member_Identifier_ 提供成员名称或成员键的有效字符串表达式。
