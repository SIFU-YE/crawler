# 算术运算符
可以在多维表达式（MDX）中使用算术运算符进行任何算术计算，包括加法、减法、乘法和除法。
MDX 支持下表中列出的算术运算符。
添加两个数字。  
---  
将一个数字除以另一个数字。  
将两个数字相乘。  
减去两个数字。  
^ （电源） | 将一个数字加到另一个数字。  
MDX 不包括用于获取数字平方根的函数。 若要获取数字的平方根，请使用 ^ 运算符将其提升为 0.5 的幂。
以下规则确定 MDX 表达式中算术运算符的优先级顺序：
  * 当表达式中有多个算术运算符时，MDX 先执行乘法和除法，然后执行减法和加法。
  * 当表达式中的所有算术运算符具有相同的优先级时，执行顺序从左到右。
  * 括号内的表达式优先于所有其他操作。
