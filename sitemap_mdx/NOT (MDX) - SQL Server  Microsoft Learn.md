# NOT (MDX)
对数值表达式执行逻辑非运算。
```
  
NOT Expression1  

```

Expression1 返回数值的有效多维表达式 (MDX) 表达式。
一个布尔值，如果参数的计算结果为 **true** ，则返回 **false** ;否则为 **true** 。
