# 函数（MDX 语法）
多维表达式 (MDX) 中有几类用于执行特定操作的内部函数。 下表列出了 MDX 中可用的函数类别。
有关各个函数的详细信息，请参阅 MDX 函数参考 (MDX) 。
数组函数 | 提供存储过程中使用的数组。 有关详细信息，请参阅 将存储过程 (MDX) 。  
---|---  
维度函数 | 从层次结构、级别或成员返回对维度的引用。 有关详细信息，请参阅 使用维度、层次结构和级别函数。  
层次结构函数 | 从级别或成员返回对层次结构的引用。 有关详细信息，请参阅 使用维度、层次结构和级别函数。  
级别函数 | 从成员、维度、层次结构或字符串表达式返回对级别的引用。 有关详细信息，请参阅 使用维度、层次结构和级别函数。  
逻辑函数 | 对对象和表达式执行逻辑操作和比较。 有关详细信息，请参阅 使用逻辑函数。  
成员函数 | 从其他对象或从字符串表达式返回对成员的引用。 有关详细信息，请参阅 使用成员函数。  
数值函数 | 对对象和表达式执行数学函数和统计函数。 有关详细信息，请参阅 使用数学函数。  
设置函数 | 从其他对象或从字符串表达式返回对集的引用。 有关详细信息，请参阅 使用集函数。  
字符串函数 | 从其他对象或从服务器返回字符串值。 有关详细信息，请参阅 使用字符串函数。  
元组函数 | 从集或从字符串表达式返回对元组的引用。 有关详细信息，请参阅“使用元组函数”。  
函数可用于或包含在任何 MDX 表达式中。 函数还可以嵌套（在一个函数内使用另一个函数）。
