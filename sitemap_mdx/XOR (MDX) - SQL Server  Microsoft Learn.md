# XOR (MDX)
对两个数值表达式执行逻辑异运算。
```
  
Expression1 XOR Expression2  
  

```

Expression1 返回数值的有效多维表达式 (MDX) 表达式。
Expression2 返回数值的有效 MDX 表达式。
一个布尔值，如果一个且只有一个参数的计算结果为 **true** ，则返回 **true** ;否则为 **false** 。
**XOR** 运算符将这两个参数视为布尔值， (零，0，为 **false** ;否则，在运算符执行逻辑排除之前，**) true** 。 下表说明了 **XOR** 运算符如何执行逻辑排除。
Expression1 | Expression2 | 返回值  
---|---|---  
true  
true | true  
true
