# MDX 语法元素 (MDX)
多维表达式 (MDX) 具有一些由大多数语句使用或受其影响的元素：
标识符是对象（如多维数据集、维度、成员和度量值）的名称。  
---  
定义单元、成员属性和单元属性中包含的数据的类型。 MDX 只支持 OLE VARIANT 数据类型。 有关 VARIANT 数据类型的强制、转换和操作的详细信息，请参阅 Platform SDK 文档中的“VARIANT 和 VARIANTARG”。  
表达式是 Analysis Services 可以解析为单个 (标量) 值或对象的语法单位。 表达式包含返回单个值、集表达式等的函数。  
运算符是与一个或多个简单的 MDX 表达式一起使用、以构成更复杂的 MDX 表达式的语法元素。  
函数是可以接受零个、一个或多个输入值并返回一个标量值或对象的语法元素。 示例包括用于添加多个值的 Sum 函数、用于从维度或级别返回一组成员的 Members 函数等。  
注释是插入到 MDX 语句或脚本中，用来说明语句用途的文本段。 Analysis Services 不执行注释 。  
保留关键字是保留起来供 MDX 使用，且不应用作 MDX 语句中使用的对象名称的关键字。  
成员、元组和集是创建 MDX 查询之前必须了解的多维数据的核心概念。
