#### 通过
# MDX 数据定义 - REFRESH CUBE
刷新多维数据集的客户端缓存。
```
  
REFRESH CUBECube_Name   

```

_Cube_Name_ 提供多维数据集名称的有效字符串表达式。
对于连接到 Analysis Services 实例的客户端应用程序，此语句会导致客户端应用程序上缓存的内存与服务器同步。 但是，通常情况下将对其进行自动检测和更新，这种情况发生之前的时间长度取决于客户端连接字符串的设置。 REFRESH CUBE 语句立即刷新数据。
对于与本地多维数据集相连的客户端应用程序，REFRESH CUBE 语句导致重新生成本地多维数据集文件。
不能刷新存储在服务器上的命名集。
