# MDX 数据操作 - CLEAR CALCULATIONS
从多维数据集中删除所有计算，并将多维数据集恢复为计算传递 0。
```
  
CLEAR CALCULATIONS [FROMCube_Expression]  

```

_Cube_Expression_ 有效的多维表达式 (MDX) 多维数据集表达式。
当多维数据集的上下文已知时（例如在 MDX 脚本中），可以省略 **FROM** 子句。
只有服务器管理员、数据库管理员或具有多维数据集中源数据访问权（即 ReadSourceData=true）的角色的成员才能执行此语句。
