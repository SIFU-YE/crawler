1 | Java API文档  | Smartbi服务器端SDK | http://wiki.smartbi.com.cn/api/javaapiv10/

## 接口依赖Jar包获取方式
以上jar包版本可能和服务器当前的版本不一致，请以服务器目前版本为准，开发时在服务器部署的smartbi.war中找文件名开头所有字母一致的jar即可。
