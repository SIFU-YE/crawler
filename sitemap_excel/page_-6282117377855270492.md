## 方法
### `getCurrentPointValueObj`
(
  * `fildAlias`

)
通过字段别名，获取当前点击对象其它列的值
#### 参数:
  * `fildAlias` String


#### 返回值:
Object: 
其它列的值对象
#### 示例:
```
function main(chartView, point, mouseEvent){//获取当前点击对象的名为产品名称的列的数据var obj = point.getCurrentPointValueObj('产品名称');
    console.log(obj);}输出：{col:0, df:null, value:"小米", sheetName:"", displayValue:"小米", row:0, realValue:""}
```

### `getMemberObj`
(
  * `memberName`

)
通过维度别名获取维度信息
#### 参数:
  * `memberName` String


#### 返回值:
Object: 
维度信息
#### 示例:
```
function main(chartView, point, mouseEvent){var obj = point.getMemberObj('产品名称');
    console.log(obj);}输出：{uniqueName:"小米", caption:"小米"}
```
