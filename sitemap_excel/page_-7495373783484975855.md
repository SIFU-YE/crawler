## 方法
### `openOlapQuery`
(
  * `callback`

)
用于打开一个新的OlapQuery对象，callback非空表示在新窗口打开
#### 参数:
  * `id` String
  * `callback` Function


### `openURL`
(
  * `postData`

)
（电子表格）打开一个URL链接
#### 参数:
  * `url` String
  * `postData` JSON
传递的数据信息, 例如：[{name:"param",value: "时间"}]
  * `name` String
  * `spec` String


#### 示例:
```
function main(spreadsheetReport, isAjaxRefreshCallback){// 在当前页面打开指定链接，以POST方法传递参数：param=时间。
    spreadsheetReport.openURL('https://www.baidu.com',[{name:"param",value:"时间"}],'_self');}
```
